About

I’m motivated by product design and real-world impact. I can (and have) spent hours ensuring my applications are pixel-perfect, and I especially enjoy turning ideas into reality. I thrive on challenges and deadlines, am fanatical about the tiny details, and believe that thorough planning can save hours of programming.

In my most recent application, Athena, I extended the React Router to create a role-based authorization system, as well as D3.js to better present the visual metric data in the dashboard. To ensure a seamless user experience, I architected the front-end to minimize expensive queries to the database and loaded the information in state using React Redux.

I particularly enjoy creating solutions to problems that I personally have. For example, I built a restaurant wish list application, because I have always struggled with the question “what should I eat for dinner?” I’ve also designed an interactive web game using the Phaser.js graphics engine, as well as a robust event manager with React Native.

I’m experienced with front-end frameworks such as React, React Native, Redux, and Angular, and comfortable with NodeJS, Mongo, and various SQL platforms. Lately, I’ve been nerding out over SASS, because I find beauty in how it adds structure to the frequent chaos of CSS.

Let's Talk

In this application, I used React/Redux, SASS, MongoDB, and PosgreSQL to develop a knowledge-base SaaS application that focuses on product analytics and customer support. By modifying the React-Router and Redux, I constructed a role-based authentication and authorization system that provided different content to the user depending on their privilege levels. Using the D3.js library, I was able to produce data visualizations and metrics to better inform decision makers of their products.

  • Category: React, Redux, SASS, MongoDB, Postgres, ElasticSearch

Flash!

Flash! is a geolocation-focused mobile app, built on React Native, for managing nearby user events. With this mobile app, a user can view publicly-hosted events nearby or can manage private events among friends. I implemented Instagram API functionality so users can get up-to-date snapshots of what is happening at the event.

Solar Measurement Tool

Solar Measurement Tool: the quick solution to the problem of solar energy estimates.

The one question that everyone raises when considering solar for their home is "How much money will I save on electricity?" I built this tool with Google Maps to allow homeowners to effectively answer that question by selecting their rooftop and entering info corresponding to their setup to analyze their savings.
