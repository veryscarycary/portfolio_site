The TL:DR

I’m motivated by product design and real-world impact. I can (and have) spent hours ensuring my applications are pixel-perfect, and I especially enjoy turning ideas into reality. I thrive on challenges and deadlines, am fanatical about the tiny details, and believe that thorough planning can save hours of programming.

In my most recent application, Reddi2Mingle, I used sockets and data caching to create a live notification system, as well as React Motion to better simulate gesture-based motions. To ensure a seamless user experience, I architected the front-end to minimize expensive queries to the database and loaded the information in state using React Redux.

I particularly enjoy creating solutions to problems that I personally have. For example, I built a restaurant recommendation engine, because I have always struggled with the question “what should I eat for dinner?” I’ve also designed an interactive web game using the PIXI graphics engine, as well as an Android chore-tracker with React Native.

I’m experienced with front-end frameworks such as React, React Native, Redux, and Angular, and comfortable with NodeJS, Mongo, Redis, and Neo4j. Lately, I’ve been nerding out over SASS, because I find beauty in how it adds structure to the frequent chaos of CSS.
